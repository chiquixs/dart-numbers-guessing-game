This repository contains a simple Number Guessing Game built using the Dart programming language. In this game, the player tries to guess a randomly generated number within a specific range

The game includes several features such as:

    - A limit on the number of attempts
    - A scoring system based on the number of guesses
    - Interactive feedback to guide the player (too high / too low)

So through this mini project, I will create 3-4 branches and make several commits to develop the features that I will create in this game

This project was developed as part of the Mobile Programming course – Week 3 assignment, focusing on the implementation of:

    - Conditional statements (branching)
    - Looping structures
    - Advanced Git operations such as branching and commit history

The repository demonstrates the development process through multiple branches and commits that show the progression of features added to the game
